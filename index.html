What is Project Midas?

MidasWWW (or Midas) for short is one of the earliest web browsers developed at the Standford Linear Accelerator Center, after the release of Berners-Lee's WWw software and library for web development (or more simply put the start of the World Wide Web). As this is a project to develop my first personal professional website, the name I think is quite self explanatory
